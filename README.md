# Netflix Pivot Dashboard Analysis

## Overview
This project analyzes Netflix content using pivot tables, charts, and slicers to understand genre trends, country contributions, and content format preferences.

## Dataset
Source: Netflix (cleaned)
Metrics engineered:
- Duration_hrs
- Freshness
- Popularity Score

## Dashboard Components
✔ Popularity by Genre
✔ Popularity by Country
✔ Popularity by Type (Movie vs TV)
✔ Slicers (Genre, Country, Type)
✔ Charts for comparison

## Insights (Business Level)
1. TV Shows have higher engagement vs Movies.
2. US dominates content contribution.
3. Crime/Thriller genres lead.
4. Fresh content increases popularity.
5. Engagement is higher for episodic formats.

## Tools Used
- Excel (Pivot Tables, Pivot Charts, Slicers)
- Windows 11
- Microsoft 365
